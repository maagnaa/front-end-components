Basic Box-Model Properties

CSS Property: box-sizing.

Div [1.1] and [1.2] have the same width, but different box-sizing.

[1.1] box-sizing: border-box

The border width does not affect the total width of the container.

[1.2] box-sizing: content-box

The border width comes in addition to the width of the container.

Borders 101

[2.1]Basic Properties

  • border-width: 1em;
  • border-style: solid;
  • border-color: #f4a261;

Also Shorthand:

  • border: 1em solid #f4a261;

[2.2] Border Radius

  • border: 1em solid #f4a261;
  • border-radius: 1.5em;

[2.3] Uneven Border Radius

  • border: 1em solid #f4a261;
  • border-radius: 0% 25%;

[2.4] Border on one side

  • border-right: 1em solid #f4a261;

[2.5] Circle!

  • border: 1em solid #f4a261;
  • border-radius: 50%;
